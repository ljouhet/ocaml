1. Installer OCaml : <http://protz.github.io/ocaml-installer/>
2. Télécharger Emacs : <http://ftp.gnu.org/gnu/emacs/windows/>  
   Décompresser le `.zip`, et lancer `emacs-.../bin/addpm.exe` pour installer Emacs.
3. Copier `tuareg.el, ocamldebug.el, default.el` dans :  
   `C:\...\emacs-...\site-lisp` (répertoire où `emacs` est *installé* ; chercher `site-lisp` dans l'explorateur...)
* Source : http://www.forge.ocamlcore.org/frs/?group_id=43
* Emplacement : https://www.gnu.org/software/emacs/manual/html_node/emacs/Init-File.html
